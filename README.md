1.  上传Bot到服务器
2.  设置伪静态（具体可以参考Thinkphp框架安装方法）

```
location / {
	if (!-e $request_filename){
		rewrite  ^(.*)$  /index.php?s=$1  last;   break;
	}
}
```
3. 选择网站运行目录为Public

4. 配置数据库（在application/database.php）

5. 导入sql到数据库
